## Usage

From the root directory:

1. Install project dependencies & Build a client bundle with Webpack:

```sh
npm install
```

2. Run schema to create `strings` database and table:

```sh
\i db/schema.sql
```

3. Start Express server:

```sh
npm start 
```

4. Open browser at `127.0.0.1:3000` 

5. Enjoy!

## Testing

```sh
npm run test:string
```
